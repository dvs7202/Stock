# Rest Live Stock api

# technologies

- Node JS
- Express JS
- Mongo DB

# prerequisites

1. Install node js in your system
2. clone repository ("https://github.com/dvs7202/Stock")
3. open live Stock api in your ide (vs code etc)
4. npm i (in terminal);

# how to run

# create .env file and add the details

PORT =

NODE_ENV =
JWT_SECRET =
JWT_COOKIE_EXPIRES_IN =

# Run (In terminal

npm run dev (or)
npm run start
